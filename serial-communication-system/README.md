# Serial Communication System

## Overview
This project explores the design and implementation of a configurable serial communication interface using digital logic principles.

After studying serial communication concepts such as UART and RS232 during university digital electronics modules, 
I wanted to independently redesign and extend the system to deepen my understanding of data framing, timing control, 
and hardware-level communication.

This project is being developed independently and is not based on assessed university coursework.

## Project Goals
- Design a serial data transmission system from digital building blocks
- Understand start/stop bit framing and timing behaviour
- Implement and test serial transmission in Multisim
- Build and validate the system on physical hardware
- Explore optimisation and modular digital system design

## Planned Features
- Serial data transmission
- Configurable data framing
- Clock and timing control
- Transmission state visualisation
- Hardware implementation and testing
